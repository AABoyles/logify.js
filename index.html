Usage

Standard Log

Try It
Code Example
logify.log("Standard log message");
Standard Log With HTML

HTML works just find in log messages. Have at it!

Try It
Code Example
var rawHTML =
  "<img src='https://placehold.it/256x128'>"+
  "<h3>This is HTML</h3>"+
  "<p>It's great, right?</p>"
logify.log(rawHTML);
Standard Log with callback

Try It
Code Example
logify.log("Standard log message with callback", function(event) {

    // The click event is in the event variable, so you can use it here.
    event.preventDefault();

    logify.log("You clicked the notification");

});

Closing Log On Click

Try It
Code Example
logify
  .closeLogOnClick(true)
  .log("Click me to close!");
Disable Log On Click

Clicking on a log message to close is disabled by default, but if you've enabled it and need to reset it to disabled, you can do so very easily.

The decision to disable it by default was to allow any type of html to be included in the log messages, including links.

Try It
Code Example
logify
  .closeLogOnClick(true)
  .log("Click me to close!")
  .closeLogOnClick(false)
  .log("You can't click to close this!");
Hide in 10 seconds

Try It
Code Example
logify
  .delay(10000)
  .log("Hiding in 10 seconds");
Persistent Log

Persistent log messages will stay until clicked or until forceable removed when the number of messages exceeds the maxLogItems setting.

Try It
Code Example
logify.log("Will stay until clicked", "", 0);
Maximum Number of Log Messages

You can easily set the maximum number of log/success/error messages that will be displayed at a single time. The default is two.

Try It
Code Example
logify
  .maxLogItems(1)
  .log("You can only see one message at a time!");

// The timeout is just for visual effect.
setTimeout(function() {
  logify.log("This message has forced the last message to close.");
}, 1000);

Theming

Themes

There is no need to insert any CSS at all to use the default theme. It's bundled with the main JavaScript file.

But Logify is easily themeable!

Theme CSS

This is the default CSS. It gets inserted before any other CSS, so any file you include (or style you put inline) can overwrite these.

  box-sizing:border-box;

  background-color:rgba(0,0,0,.6);
  left:0;
  right:0;

  height:100%;
  z-index:99999;
}

Performance

Logify is designed from the ground-up for great performance and a small footprint. That means it's not as feature-rich as some other options. That's on purpose. We don't plan on adding a lot more features so it stays a great option for every kind of website and user.

Hook it up!

Notes

Obviously, if you're installing via Bower or NPM, you'll need to include the files in the "dist" directory in your HTML for everything to work. But you knew that already.

It's also worth noting that the CSS is bundled by default, so there's no need to include any stylesheets to use the default theme. It's dyamically inserted before any other <link> elements (like) stylesheets so it's super easy to override with your own styles.

AngularJS

An AngularJS module is also included. It's in the "ngLogify.js" file, so use that instead if you're building an AngularJS app.

Initialize ngLogify

// Make sure to include the ngLogify.js file.
angular
    .module("myApp", ["ngAlerify"])
    .controller("myController", function($scope, ngLogify) {
        ngLogify.success("Welcome to ngLogify!");
    });
